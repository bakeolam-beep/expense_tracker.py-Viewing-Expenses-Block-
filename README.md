# Expense Tracker (Python CLI)

A simple command-line expense tracker built with Python and CSV storage.

## Features
- Add expenses
- View all expenses
- Persistent CSV storage

## Viewing Expenses
The "View Expenses" feature reads from `expenses.csv`, skips the header, and prints all stored expenses in a readable table format.

## Requirements
- Python 3.x

## Run
```bash
python expense_tracker.py
